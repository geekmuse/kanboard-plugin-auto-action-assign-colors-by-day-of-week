## Codebase Patterns
- **getParam API**: `$this->getParam($key)` returns the configured value for an action parameter — ActionManager pre-populates `$this->params` from `action_has_params` before `execute()` is called. Never query the DB to fetch action params.
- **Parameter key strategy (FIXED)**: `getActionRequiredParameters()` keys MUST be fixed English strings (e.g., `'Monday'`), never wrapped in `t()`. `DateTime::format('l')` always returns English day names. Using `t()` on keys stores translated names in `action_has_params` that can never be matched by the English lookup — permanent failure in non-English installs.
- **PSR-12 braces**: PHP classes and methods require opening brace on its own line (Allman style). Use `cytopia/phpcs:latest` Docker image to verify PSR-12 compliance when vendor/ is not yet bootstrapped.
- **docker-*.sh skip pattern**: Scripts in `scripts/` exit 0 with a SKIP message when vendor/ is missing. This allows pre-commit hooks to pass during early development before US-010 bootstraps composer/vendor.
